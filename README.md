## Getting Started

Install dependencies:

```bash
yarn install
# or
npm install
```

Setup environment variables:

```bash
cp .env.example .env.local
```

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
